# Cortex v1

A deterministic geometric controller driving a stochastic information filter.

## What it is

A 3×3×3 cubic lattice (24 rotation symmetries) coupled to a Matrix Product State quantum simulator via a polarity governor. Text is processed word-by-word: each word maps to a rotation, which produces a geometric signal (α) that modulates the MPS entropy ceiling. High-α words get more room to build entanglement; low-α words get pruned harder.

## Architecture

```
word
  → semantic bridge (GloVe-50 → PCA-3D → axis/angle → SO(3) → SU(2))
  → α = mean |cos θ| over lattice node displacements
  → polarity governor: effective_S_max = S_max_base × (1 + α × polarity)
  → MPS simulator: apply SU(2) gate, CNOT, SVD truncation under governor ceiling
```

## Measured results

Run `python livnium_cortex_v1.py` for the full 12-invariant self-test. Run `python benchmark/benchmark_retrieval_triage.py` for the retrieval benchmark.

### MPS simulator scaling

| Qubits | Circuit | max χ | Time | Memory | Measurement |
|--------|---------|-------|------|--------|-------------|
| 5 | GHZ | 2 | 0.002s | 512B | 100/100 valid |
| 100 | GHZ | 2 | 0.001s | 12.6KB | 100/100 valid |
| 500 | GHZ | 2 | 0.006s | 63.9KB | 100/100 valid |
| 1000 | GHZ | 2 | — | 127.9KB | 100/100 valid |
| 10 | random 5-layer | 32 | — | 43.6KB | trunc_err = 0.0 |
| 20 | random 5-layer | 64 (cap) | — | 1.2MB | trunc_err = 10.2 |
| 50 | random 5-layer | 64 (cap) | — | 5.2MB | trunc_err = 10.2 |

GHZ states scale to 1000+ qubits because they are area-law: bond dimension stays at χ = 2 regardless of n. Random deep circuits hit the χ cap at n ≈ 20 and accumulate truncation error. This is expected MPS behavior — the simulator is exact for structured entanglement and approximate for volume-law entanglement.

| Qubits | max χ | Prune events | Mean entropy | Mean polarity | Trunc err | Memory |
|--------|-------|-------------|-------------|---------------|-----------|--------|
| 15 | 4 | 0 | 0.297 | 0.851 | 0.0 | 3.3KB |
| 100 | 4 | 0 | 0.336 | 0.967 | 0.0 | 25KB |
| 500 | 4 | 0 | 0.344 | 0.990 | 0.0 | 127KB |
| 1000 | 4 | 0 | 0.346 | 0.994 | 0.0 | 255KB |

At high α and high polarity, the governor's effective ceiling is generous enough that no truncation occurs. The MPS stays in the area-law regime. Memory scales linearly.

Note: the default organism uses MD5 → rotation mapping, which is arbitrary (no semantic relationship to word meaning). Prune/survive decisions are driven by bond entropy thresholds, not word importance. The semantic bridge (`--mode real`) replaces MD5 with GloVe embeddings for content-aware filtering.

### α-triage retrieval benchmark

Memory capped at 40% of token count. Ground truth: 10 factual tokens from a passage about Shor's algorithm.

| Policy | Fact recall |
|--------|------------|
| FIFO | 30% |
| LRU | 30% |
| α-triage (mock) | 100% |

α-triage retains all 10 ground-truth facts. FIFO/LRU retain only the 3 facts that happen to appear in the last 40% of the passage. The mock α table assigns scores by hand; real mode uses GloVe-derived α via the semantic bridge.

## What it does not do

- The MPS state is a quantum state, not a word store. It does not "remember" words.
- The organism's prune/survive split is driven by bond entropy, not semantic understanding.
- The MD5 → rotation mapping is deterministic but arbitrary. Only the semantic bridge (GloVe mode) connects word meaning to lattice geometry.
- The polarity governor is an "entanglement counsellor, not a dictator" — when bond dimension hits the hard χ cap, the governor can relax its entropy ceiling but cannot create more physical space.
- GHZ scaling to 1000 qubits is real but expected — GHZ is the easiest entangled state for MPS (area-law, χ = 2 always). Volume-law states do not scale.

## Running

```bash
# Self-test (no dependencies beyond numpy)
python cortex_v1/livnium_cortex_v1.py

# Benchmark (mock mode, no dependencies)
python cortex_v1/benchmark/benchmark_retrieval_triage.py

# Benchmark (real mode, requires gensim + scikit-learn)
python cortex_v1/benchmark/benchmark_retrieval_triage.py --mode real

# Organism (MD5 mode)
python -c "
from cortex_v1.organism_seed import ChatOrganism
org = ChatOrganism(n_qubits=15)
org.feed('your text here')
org.audit()
"
```
